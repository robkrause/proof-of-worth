# Proof of Worth

> Products these days are often of low quality. We call them Fiat Products. This site lists products that are outstanding and give us a glimpse of what every product would be like in an Bitcoin Standard.

https://proof-of-work.netlify.app/

### How to run

* `npm i`
* `npm run dev`

## Contribute products

* add product in `src\data\directory\directory.json`
* add image in `src\data\directory\directory.json`

Discussion/vote can be done in pull request
